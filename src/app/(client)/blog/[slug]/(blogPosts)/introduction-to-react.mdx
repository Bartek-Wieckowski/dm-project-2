# Introduction to React

React is a JavaScript library for building user interfaces. It is maintained by Facebook and a community of individual developers and companies.

## Why React?

- **Component-Based**: Build encapsulated components that manage their own state, then compose them to make complex UIs.
- **Declarative**: React makes it painless to create interactive UIs. Design simple views for each state in your application.
- **Learn Once, Write Anywhere**: You can develop new features in React without rewriting existing code.

### Getting Started

To create a new React application, you can use the Create React App tool:

```bash
npx create-react-app my-app
cd my-app
npm start
```

## React

React to biblioteka JavaScript, która umożliwia tworzenie interfejsów użytkownika w formie komponentów. Oto przykład prostego komponentu React:

```jsx
// Przykład komponentu React
import React from 'react';

const Hello = ({ name }) => {
  return <div>Hello, {name}!</div>;
};

export default Hello;
```


## HTML

HTML (HyperText Markup Language) jest językiem znaczników używanym do budowy struktur stron internetowych. Oto przykład prostego kodu HTML:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <h1>Hello, world!</h1>
  <p>This is a paragraph.</p>
</body>
</html>
```


## CSS

CSS (Cascading Style Sheets) jest językiem arkuszy stylów używanym do stylizacji stron internetowych. Oto przykład prostego stylu CSS:

```css
/* Przykład stylu CSS */
body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
}

.container {
  max-width: 800px;
  margin: 0 auto;
}

h1 {
  color: blue;
}

p {
  font-size: 16px;
  line-height: 1.5;
}
```
